Executive Summary

Modern software development is moving rapidly with generative AI, yet financial institutions are often restricted from using public cloud-hosted code assistants due to strict regulatory rules (e.g., GDPR, PCI-DSS, local banking guidelines). TemenAI solves this friction. By running entirely offline inside the bank's corporate network and interfacing with secure local LLMs, TemenAI enables T24 developers to customize banking routines safely, maintaining 100% code privacy.

Why Financial Institutions Choose TemenAI

A side-by-side analysis of typical developer bottlenecks and how TemenAI addresses them.

The Cloud Security Barrier

Cloud-based assistants stream proprietary financial logic and database structure schemas to external servers, risking regulatory non-compliance.

100% On-Premises Privacy

TemenAI integrates with models hosted in your private datacenter (Ollama, vLLM). No code or workspace context ever leaves the bank's secure network.

Generic Model Limitations

Standard models lack training on InfoBasic (the proprietary language of Temenos T24) and T24-specific APIs like F.READ, OPF, or common blocks.

Domain-Specific Expertise

TemenAI supports fine-tuned model backends specialized in InfoBasic syntax and Java workspace structures, providing highly accurate validation, version, and enquiry code.

Eclipse / Design Studio Isolation

Most modern AI coding assistants focus exclusively on VS Code or IntelliJ, leaving Eclipse and Temenos Design Studio developers isolated without AI features.

Native IDE Integration

TemenAI is built as a native Eclipse Plug-in, guaranteeing seamless installation and UI integration directly within Eclipse and Temenos Design Studio.

Features Designed for Developer Velocity

A breakdown of the developer workflows and UI integrations provided by the TemenAI plugin.

Inline Code Generation

Place your cursor inside any Java or InfoBasic source file and press Ctrl+Alt+A. TemenAI collects context around your cursor and streams raw, syntactically correct completions directly into the editor.

Interactive Chat Sidebar

Open the chat sidebar under Window > Show View. Ask architectural questions, write unit tests, or explain routines. Any text selected in your active code editor is automatically included in your query as markdown context.

Agentic Workspace Tool Calling

TemenAI can read, write, and list workspace directories via standard tool execution blocks. The AI can inspect structural templates, reference library files, and write customized validation routines recursively.
